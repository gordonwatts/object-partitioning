# 5% max partition / ~1% min group report

## Introduction

We want bin boundaries for the dataset
`data18_13TeV:data18_13TeV.periodAllYear.physics_Main.PhysCont.DAOD_PHYSLITE.grp18_v01_p6697`
that keep the largest partition near 5% while avoiding ultra-sparse bins. All
runs use `-n 50` and `--ignore-axes met` to keep MET out of the binning.

## Baseline binning (bins-per-axis = 3)

This is the best baseline from the bins-per-axis tuning study (max fraction
0.039, zero bins 0). The algorithm computes per-axis percentile-based
boundaries (a percentile boundary is a value where a fixed fraction of events
fall below it)
so each axis has roughly equal event counts per bin. It then builds the full
n-D histogram by crossing those per-axis bins without any post-processing.

Command:

```bash
atlas-object-partitioning partition \
  data18_13TeV:data18_13TeV.periodAllYear.physics_Main.PhysCont.DAOD_PHYSLITE.grp18_v01_p6697 \
```

## Target scan for bins-per-axis

The target scan searched bins-per-axis 2-4 with min/max goals 0.01/0.05. For
each candidate bins-per-axis value, it first recomputes the per-axis quantile
boundaries using that bin count, then builds the n-D histogram from those
boundaries and records the max bin fraction and minimum bin fraction. It then
computes how far each candidate is above the max target and below the min
target, adds those two gaps together, and selects the candidate with the
smallest total gap (breaking ties by smaller max fraction and larger min
fraction).

Command:

```bash
atlas-object-partitioning partition \
  data18_13TeV:data18_13TeV.periodAllYear.physics_Main.PhysCont.DAOD_PHYSLITE.grp18_v01_p6697 \
  -n 50 --ignore-axes met \
  --target-min-fraction 0.01 --target-max-fraction 0.05 \
```

## Per-axis sparse-bin merging

Merging sparse bins along each axis did not trigger any merges at the 1%
threshold. For each axis, the algorithm computes 1-D counts by summing the full
n-D histogram over all *other* axes (so each axis bin gets the total events in
all cells that share that axis bin). It then repeatedly merges the smallest
adjacent axis bins until the minimum fraction threshold is met or the axis hits
a minimum bin count.

```bash
atlas-object-partitioning partition \
```
